This folder is for custom favicons. Remove these placeholder files and add your own favicon files using these exact filenames if you want to replace the site's favicons.

Required filenames (if you upload your own):
- apple-touch-icon.png
- favicon-32x32.png
- favicon-16x16.png
- favicon.ico
- android-chrome-192x192.png
- android-chrome-512x512.png
- mstile-150x150.png
- site.webmanifest
- browserconfig.xml

Placing your custom files with these names will replace the default favicons generated in the built site. They will be copied to the `_site` folder on build.
